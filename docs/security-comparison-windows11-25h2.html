Как открыть в Microsoft Word:
Способ 1 (рекомендуется): Откройте Word → Файл → Открыть → выберите этот файл (security-comparison-windows11-25h2.html). Word автоматически отобразит таблицу.
Способ 2: Перетащите файл на значок Word — он откроется напрямую как документ с таблицей.
Способ 3: Откройте файл в браузере → выделите всю таблицу (Ctrl+A) → скопируйте (Ctrl+C) → вставьте в Word (Ctrl+V). Форматирование таблицы сохранится.

После открытия: Файл → Сохранить как → Word Document (.docx), чтобы сохранить в формате Word.
